# Coastal Roof Cleaning Experts — Website Redesign Concept

This is a static website redesign concept for Coastal Roof Cleaning Experts.

## Files

- `index.html` — Main website file
- `assets/` — Optimized local image assets
- `pitch-email.md` — Sales pitch email
- `pricing.md` — Recommended public-facing pricing
- `image-notes.md` — Image documentation
- `image-manifest.json` — Source-to-asset image manifest

## GitHub Pages Setup

1. Create a new GitHub repository.
2. Upload everything inside this folder to the repository root.
3. Go to repository Settings.
4. Open Pages.
5. Under Build and deployment, select:
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/root`
6. Save.
7. GitHub will provide a live URL after deployment.

## Notes

The forms currently use `mailto:` as a prototype. For a production site, connect the forms to a backend or service such as Formspree, Netlify Forms, HubSpot, Zapier, or a CRM.

The images in `assets/` are optimized local JPGs so the website does not depend on remote image URLs.
